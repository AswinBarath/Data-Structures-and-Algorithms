## Plan to learn Data Structures and Algorithms

### Techniques for efficient learning

‚û° **Take notes** on each and every single concept you learn

‚û° **Active Recall** is the best technique to learn

### Framework

- ‚û° **Theory** of basics from each Data Structure and Algorithm
- ‚û° **Code Implementation** of each Data Structure and Algorithm
- ‚û° **Curate Leetcode Interview Problems** and sort them by:
    - Data Structure/Algorithm used
    - Techniques used
    - Optimization used

### Advice based on experience

‚û° Start solving problems on your own with regular breaks

‚û° Use online resources only when necessary and save them in the respective GitHub Repositories

‚û° Use Git and GitHub with Habit Tracker for Dopamine Hits & Progress Visibility
