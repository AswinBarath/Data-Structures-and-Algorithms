Rectangle Area (LeetCode 223)

Given the coordinates of two axis-aligned rectangles in a 2D plane, return the total area covered by the two rectangles.

The first rectangle is defined by its bottom-left corner (ax1, ay1) and its top-right corner (ax2, ay2).

The second rectangle is defined by its bottom-left corner (bx1, by1) and its top-right corner (bx2, by2).

Constraints:

  • -104 ‚â§ ax1 ‚â§ ax2 ‚â§ 104
  • -104 ‚â§ ay1 ‚â§ ay2 ‚â§ 104
  • -104 ‚â§ bx1 ‚â§ bx2 ‚â§ 104
  • -104 ‚â§ by1 ‚â§ by2 ‚â§ 104

Examples:

Example 1:
Input: ax1 = -3, ay1 = 0, ax2 = 3, ay2 = 4, bx1 = 0, by1 = -1, bx2 = 9, by2 = 2
Output: 45
Example 2:
Input: ax1 = -2, ay1 = -2, ax2 = 2, ay2 = 2, bx1 = -2, by1 = -2, bx2 = 2, by2 = 2
Output: 16

Key Insight:

The total area covered by two rectangles is:

Total Area = Area of Rectangle A + Area of Rectangle B - Overlap Area

We subtract the overlap area because it would be counted twice if we just added the areas of both rectangles.

üí° Solution Explained

Step-by-Step Solution

Step 1: Calculate Individual Rectangle Areas

For any rectangle with corners (x1, y1) and (x2, y2):

Area = (x2 - x1) √ó (y2 - y1)

Step 2: Find Overlap Dimensions

Two rectangles overlap if:

  • They overlap horizontally: max(ax1, bx1) < min(ax2, bx2)
  • They overlap vertically: max(ay1, by1) < min(ay2, by2)

If they overlap, the overlap rectangle has:

  • Left edge: max(ax1, bx1)
  • Right edge: min(ax2, bx2)
  • Bottom edge: max(ay1, by1)
  • Top edge: min(ay2, by2)

Step 3: Calculate Overlap Area

Overlap Width = max(0, min(ax2, bx2) - max(ax1, bx1))

Overlap Height = max(0, min(ay2, by2) - max(ay1, by1))

Overlap Area = Overlap Width √ó Overlap Height

Note: We use max(0, ...) to handle cases where rectangles don't overlap.

Step 4: Apply Formula

Total Area = Area A + Area B - Overlap Area

Example Walkthrough

Example: Rectangle A: (-1, -1) to (1, 1), Rectangle B: (0, 0) to (2, 2)

Step 1: Individual Areas
‚Ä¢ Area A = (1 - (-1)) √ó (1 - (-1)) = 2 √ó 2 = 4
‚Ä¢ Area B = (2 - 0) √ó (2 - 0) = 2 √ó 2 = 4

Step 2: Overlap Bounds
‚Ä¢ Left: max(-1, 0) = 0

‚Ä¢ Bottom: max(-1, 0) = 0
‚Ä¢ Top: min(1, 2) = 1

‚Ä¢ Width: max(0, 1 - 0) = 1

‚Ä¢ Total = 4 + 4 - 1 = 7

Complete Java Implementation

class Solution {
/**
* Computes the total area covered by two axis-aligned rectangles.
*
* @param ax1 x-coordinate of the bottom-left corner of rectangle A
* @param ay1 y-coordinate of the bottom-left corner of rectangle A
* @param ax2 x-coordinate of the top-right corner of rectangle A
* @param ay2 y-coordinate of the top-right corner of rectangle A
* @param bx1 x-coordinate of the bottom-left corner of rectangle B
* @param by1 y-coordinate of the bottom-left corner of rectangle B
* @param bx2 x-coordinate of the top-right corner of rectangle B
* @param by2 y-coordinate of the top-right corner of rectangle B
* @return total area covered by both rectangles
*/
public int computeArea(int ax1, int ay1, int ax2, int ay2,
int bx1, int by1, int bx2, int by2) {

// Step 1: Calculate individual rectangle areas
int areaA = (ax2 - ax1) * (ay2 - ay1);
int areaB = (bx2 - bx1) * (by2 - by1);

// Step 2: Calculate overlap dimensions
int overlapWidth = Math.max(0, Math.min(ax2, bx2) - Math.max(ax1, bx1));
int overlapHeight = Math.max(0, Math.min(ay2, by2) - Math.max(ay1, by1));

// Step 3: Calculate overlap area
int overlapArea = overlapWidth * overlapHeight;

// Step 4: Return total area (avoiding double counting)
return areaA + areaB - overlapArea;
}
}

Alternative Implementation with Helper Method

class Solution {

/**
* Computes the total area covered by two axis-aligned rectangles.
* Uses helper methods for better code organization and readability.
*
* @param ax1 x-coordinate of the bottom-left corner of rectangle A
* @param ay1 y-coordinate of the bottom-left corner of rectangle A
* @param ax2 x-coordinate of the top-right corner of rectangle A
* @param ay2 y-coordinate of the top-right corner of rectangle A
* @param bx1 x-coordinate of the bottom-left corner of rectangle B
* @param by1 y-coordinate of the bottom-left corner of rectangle B
* @param bx2 x-coordinate of the top-right corner of rectangle B
* @param by2 y-coordinate of the top-right corner of rectangle B
* @return total area covered by both rectangles
*/
public int computeArea(int ax1, int ay1, int ax2, int ay2,
                          int bx1, int by1, int bx2, int by2) {

int areaA = calculateRectangleArea(ax1, ay1, ax2, ay2);
int areaB = calculateRectangleArea(bx1, by1, bx2, by2);
int overlapArea = calculateOverlapArea(ax1, ay1, ax2, ay2,
                                              bx1, by1, bx2, by2);

return areaA + areaB - overlapArea;
}

/**
* Calculates the area of a single rectangle.
*
* @param x1 x-coordinate of bottom-left corner
* @param y1 y-coordinate of bottom-left corner
* @param x2 x-coordinate of top-right corner
* @param y2 y-coordinate of top-right corner
* @return area of the rectangle
*/
private int calculateRectangleArea(int x1, int y1, int x2, int y2) {
return (x2 - x1) * (y2 - y1);
}

/**
* Calculates the overlap area between two rectangles.
*
* @param ax1 x-coordinate of bottom-left corner of rectangle A
* @param ay1 y-coordinate of bottom-left corner of rectangle A
* @param ax2 x-coordinate of top-right corner of rectangle A
* @param ay2 y-coordinate of top-right corner of rectangle A
* @param bx1 x-coordinate of bottom-left corner of rectangle B
* @param by1 y-coordinate of bottom-left corner of rectangle B
* @param bx2 x-coordinate of top-right corner of rectangle B
* @param by2 y-coordinate of top-right corner of rectangle B
* @return overlap area between the two rectangles
*/
private int calculateOverlapArea(int ax1, int ay1, int ax2, int ay2,
                                   int bx1, int by1, int bx2, int by2) {
// Find overlap bounds
int left = Math.max(ax1, bx1);
int right = Math.min(ax2, bx2);
int bottom = Math.max(ay1, by1);
int top = Math.min(ay2, by2);

// Check if there's actually an overlap
if (left >= right || bottom >= top) {
            return 0; // No overlap
}

return (right - left) * (top - bottom);
}
}

Time & Space Complexity

Time Complexity: O(1)

All operations are constant time arithmetic calculations.

Space Complexity: O(1)

Only using a constant amount of extra space for variables.

Key Points to Remember

  • Integer Overflow: Be careful with large coordinates that might cause overflow when multiplied.
  • No Overlap Case: When rectangles don't overlap, Math.max(0, negative_value) ensures overlap area is 0.
  • Edge Cases: Handle cases where rectangles just touch (no actual overlap area).
  • Coordinate System: Remember that (x1, y1) is bottom-left and (x2, y2) is top-right.
